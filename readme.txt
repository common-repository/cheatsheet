Cheatsheet

== Description ==

This is a quick and easy place to save tips and tricks for Wordpress administrators.  After launching a new website, web designers and developers can leave notes for Wordpress administrators to reference when updating the site.  The Cheatsheet link appears under the Appearance panel.

== Installation ==

1. Upload the plugin's folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
